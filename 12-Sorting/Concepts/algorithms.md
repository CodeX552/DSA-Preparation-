# Sorting Algorithms
## O(n²): Bubble (swap adjacent), Selection (find min), Insertion (insert in sorted)
## O(n log n): Merge Sort (divide+merge), Quick Sort (partition+recurse)
## O(n+k): Counting Sort (range known), Radix Sort (digit by digit)
## Interview mein: Merge Sort aur Quick Sort code likhna aana chahiye!
