# 🔢 Array — Start Here!

## Topic Kya Hai?
Array ek **contiguous memory block** hai jisme same type ke elements store hote hain. Ye DSA ka **sabse basic aur sabse important** topic hai — almost har interview mein array questions aate hain.

## 📊 Interview Importance: 🔴 95% interviews mein puchha jaata hai

## 🎯 Kya Seekhna Hai?
1. Array traversal aur manipulation
2. Two Pointer technique
3. Sliding Window pattern
4. Kadane's Algorithm (Maximum Subarray)
5. Prefix Sum technique
6. Dutch National Flag Algorithm
7. Matrix operations
