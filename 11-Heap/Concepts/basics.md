# Heap Basics + Patterns
## Min Heap: Parent <= Children. Max Heap: Parent >= Children.
## PriorityQueue default = Min Heap. Collections.reverseOrder() = Max Heap.
## Top-K: Keep K elements in heap, poll extras. Peek = Kth element.
## Two Heaps: Median finder — left (maxHeap) + right (minHeap), balance sizes.
