# 📝 String — Start Here!

## Topic Kya Hai?
String ek **sequence of characters** hai. Java mein Strings **immutable** hote hain (ek baar bana toh change nahi hota). String problems interviews mein bahut common hain — pattern matching, manipulation, aur sliding window techniques yahan use hoti hain.

## 📊 Interview Importance: 🔴 85% interviews mein puchha jaata hai

## 🎯 Kya Seekhna Hai?
1. String manipulation (reverse, rotate, compare)
2. Palindrome detection patterns
3. Anagram problems
4. Sliding Window on strings
5. Two Pointer on strings
6. StringBuilder vs String (performance)
7. Character frequency counting
