# 📚 Stack — Start Here!

## Topic Kya Hai?

## 📊 Interview Importance: 🟠 70% interviews mein puchha jaata hai

1. Stack operations (push, pop, peek)
2. Parenthesis matching problems
3. Monotonic Stack (NGE/NSE patterns)
4. Min Stack design
5. Expression evaluation
6. Stock Span / Daily Temperatures pattern
