# Stack Basics Concept
## LIFO — Last In First Out
Push = top pe daalo, Pop = top se nikalo. Java mein `Deque<Integer> stack = new ArrayDeque<>()` use karo. Legacy `Stack` class avoid karo (synchronized, slow).

## Stack Applications
- Function call stack (recursion)
- Undo/Redo operations
- Browser back button
- Expression evaluation
- Parenthesis matching
