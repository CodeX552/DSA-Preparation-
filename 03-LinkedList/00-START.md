# 🔗 Linked List — Start Here!

## Topic Kya Hai?
Linked List ek **linear data structure** hai jisme elements **nodes** mein store hote hain. Har node mein **data + next pointer** hota hai. Array se different hai — memory mein contiguous nahi hota, dynamic size hai.

## 📊 Interview Importance: 🟠 80% interviews mein puchha jaata hai

## 🎯 Kya Seekhna Hai?
1. Singly Linked List operations (insert, delete, traverse)
2. Reverse Linked List (iterative + recursive)
3. Two Pointer / Fast-Slow technique (Floyd's)
4. Merge two sorted lists
5. Cycle detection & removal
6. Doubly Linked List basics
