# 🔗 Two Pointer Pattern on Linked List

## Pattern Types

### 1. Fast-Slow (Tortoise & Hare)
- **Middle element**: slow 1x, fast 2x
- **Cycle detection**: slow meets fast = cycle
- **Nth from end**: fast N steps ahead, then both 1 step

### 2. Two Independent Pointers
- **Intersection**: switch heads, same total distance
- **Merge sorted**: compare and advance smaller

## Key Insight
Linked List mein index nahi hota, toh two pointer se distance relationships exploit karte hain.
