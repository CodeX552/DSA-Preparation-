# 🌳 Tree — Start Here!

## Topic Kya Hai?
Tree ek **hierarchical data structure** hai jisme nodes parent-child relationship mein hote hain. **Binary Tree** mein har node ke maximum 2 children hote hain. Ye DSA ka **MOST IMPORTANT** topic hai — 95% interviews mein puchha jaata hai.

## 📊 Interview Importance: 🔴 95% interviews — SABSE ZYADA IMPORTANT!

## 🎯 Kya Seekhna Hai?
1. Binary Tree basics + Node structure
2. Traversals (Inorder, Preorder, Postorder, Level-order)
3. BST (Binary Search Tree) operations
4. Tree patterns (Height, Diameter, LCA, Views)
5. Recursive thinking on trees
