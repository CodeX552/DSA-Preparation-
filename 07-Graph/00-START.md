# 🕸️ Graph — Start Here!

## Topic Kya Hai?
Graph ek **non-linear data structure** hai jisme **nodes (vertices)** aur **edges** hote hain. Nodes ko edges connect karte hain. Real-life examples: social networks, maps, web pages.

## 📊 Interview Importance: 🔴 85% interviews mein — Google/Amazon mein MUST

## 🎯 Kya Seekhna Hai?
1. Graph representation (Adjacency List, Matrix)
2. BFS & DFS traversal
3. Cycle detection
4. Shortest path (Dijkstra, BFS)
5. Topological Sort
6. Connected components
